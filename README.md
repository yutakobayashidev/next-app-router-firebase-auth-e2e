# next-app-router-firebase-auth-e2e

Implementation with a minimal E2E test for Next.js App Router, Firebase Local Emulator, and session-based authentication.

```bash
# install deps
$ pnpm i

# create .firebaserc
$ pnpm firebase init

# run dev server
$ pnpm dev

# run e2e
$ pnpm build:test
$ pnpm test:e2e
```
